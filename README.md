# 🚇 Metro Ticket Booking System (ServiceNow)

## 📌 Project Overview
This project is a Metro Ticket Booking System developed using ServiceNow. It allows users to:
- Select source and destination
- Automatically calculate fare
- Make payment
- Generate QR code for ticket validation

---

## ⚙️ Features
- Dynamic fare calculation using GlideAjax
- Payment simulation using UI Action
- QR Code generation after payment
- Conditional UI (hide/show QR)
- Clean workflow from booking → payment → ticket

---

## 🏗️ Technologies Used
- ServiceNow
- GlideAjax
- Script Include
- Catalog Client Scripts
- UI Policies
- UI Actions

---

## 🔄 Workflow
1. User selects stations
2. Fare auto-calculates
3. Click "Pay Now"
4. Status changes to Paid
5. QR Code generated


## 🚀 How to Run
1. Import/update scripts in ServiceNow
2. Configure tables and fields
3. Open Record Producer
4. Book ticket and test flow

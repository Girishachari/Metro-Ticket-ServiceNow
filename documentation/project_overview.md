# Project Overview

The Metro Ticket Booking System is designed to simulate a real-world metro ticket booking application using ServiceNow.

## Objectives

* Automate ticket booking process
* Calculate fare dynamically
* Simulate payment system
* Generate QR-based ticket

## Modules

* Booking Module
* Fare Calculation Module
* Payment Module
* QR Generation Module

## Outcome

A fully functional workflow from booking to QR-based ticket generation.
